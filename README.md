# 제피라이크 (ZEPHIRIKE)

Retro Arcade Fantasy RPG 스타일의 웹 기반 탑다운 액션 게임.

## 🎮 게임 플레이

- **WASD**: 캐릭터 이동
- **마우스**: 공격 방향 조준
- **TAB**: 공격 모드 전환 (마우스 조준 / 자동 조준)
- **스페이스바**: 게임 시작

## 🚀 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 시작
npm run dev
```

## 🛠️ 기술 스택

- **Phaser 3.80+**: 2D 게임 엔진
- **TypeScript**: 타입 안전한 개발
- **Vite**: 빠른 개발 서버 & 빌드 도구

## 📁 프로젝트 구조

```
src/
├── entities/       # 플레이어, 적, 투사체 등 게임 객체
├── scenes/        # 메뉴, 게임 등 씬
├── systems/       # 무기, 스폰, 파티클 등 시스템
├── ui/            # HUD, 모달 등 UI 컴포넌트
└── types/         # 타입 정의
```

## 🎯 게임 특징

- 다양한 무기 시스템 (근접, 원거리, 마법)
- 웨이브 기반 적 스폰
- 레벨업 및 경험치 시스템
- 동적 파티클 효과
- 반응형 HUD
